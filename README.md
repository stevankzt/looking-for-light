# looking-for-light
Looking for Light — двухмерный рогалик с видом сверху, разработанный на платформе MonoGame (C#). Игрок оказывается в череде мрачных, процедурно-наполненных подземелий и должен пробиться сквозь пять уровней нарастающего ужаса, чтобы достичь тёплой, светлой двери — единственного выхода с каждого этажа. . Всё действие оказывается страшным сном.
